# 海龟交易法则选股器 (Turtle Trading Screener)

基于经典海龟交易法则的 A 股选股工具，结合 A 股市场特点进行优化。

## 核心规则

### 1. 股价创 20 日新高 (Breakout)
- 当前收盘价超过过去 20 个交易日的最高价
- 这是海龟交易的核心突破信号

### 2. ATR 低位震荡后向上 (Volatility Expansion)
- **ATR (Average True Range)**: 平均真实波幅，衡量股价波动率
- **低位震荡**: ATR 变异系数小于 5%，表示波动率处于稳定状态
- **向上**: 当前 ATR 较前 5 日均值上升超过 10%
- 这表示股价结束盘整，开始新的趋势

### 3. 基于 2% 风险敞口的头寸计算 (Position Sizing)
- 每笔交易最大风险：总资金的 2%
- 止损距离：2N (2 倍 ATR)
- 建仓股数 = 风险金额 / 止损距离
- A 股最小交易单位：100 股

## 使用方法

### 扫描默认股票池
```bash
python -m app.turtle_screener
```

### 检查单只股票
```bash
python -m app.turtle_screener 600519
```

### 使用自定义股票池
```bash
python -m app.turtle_screener --pool
```

### 指定资金量
```bash
python -m app.turtle_screener --capital 200000
```

### 使用缓存数据（不下载）
```bash
python -m app.turtle_screener --no-download
```

### 显示详细信息
```bash
python -m app.turtle_screener 600519 --verbose
```

## 头寸计算公式

```
风险金额 = 总资金 × 2%
止损距离 = ATR × 2
建仓股数 = 风险金额 / 止损距离
建仓金额 = 建仓股数 × 当前价格
止损价格 = 当前价格 - 止损距离
```

### 示例
- 总资金：¥100,000
- 当前股价：¥50.00
- ATR：2.50

计算：
- 风险金额 = 100,000 × 2% = ¥2,000
- 止损距离 = 2.50 × 2 = ¥5.00
- 建仓股数 = 2,000 / 5.00 = 400 股
- 建仓金额 = 400 × 50 = ¥20,000
- 止损价格 = 50.00 - 5.00 = ¥45.00

## 配置参数

可以通过 `TurtleConfig` 类自定义参数：

```python
from app.core.turtle import TurtleScreener, TurtleConfig

config = TurtleConfig(
    breakout_window=20,          # 突破周期 (默认 20)
    atr_period=14,               # ATR 周期 (默认 14)
    atr_consolidation_threshold=0.05,  # ATR 震荡阈值 (默认 5%)
    atr_rising_threshold=0.1,    # ATR 上升阈值 (默认 10%)
    risk_per_trade=0.02,         # 每笔风险 (默认 2%)
    stop_loss_multiple=2.0,      # 止损倍数 (默认 2N)
    min_capital=100000,          # 最小资金 (默认 10 万)
)

screener = TurtleScreener(config)
```

## 注意事项

1. **数据质量**: 确保有足够的历史数据（至少 60 个交易日）
2. **市场状态**: 海龟交易适合趋势市，震荡市可能产生虚假信号
3. **风险控制**: 2% 风险是上限，可根据个人风险偏好调整
4. **止损执行**: 严格执行止损，这是海龟交易的核心原则
5. **头寸调整**: 股价大幅波动后应重新计算头寸
